docker run -d --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:4-management

-d: Roda o container em segundo plano (detached mode).
--name rabbitmq: Define um nome amigável para o container.
-p 5672:5672: Porta para comunicação com o broker (AMQP).
-p 15672:15672: Porta da interface gráfica de gerenciamento.
rabbitmq:4-management: Imagem oficial que já vem com o plugin de gerenciamento ativado por padrão.
